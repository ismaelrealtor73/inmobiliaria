## 1. Estructura del equipo y roles

| Rol | Responsabilidades |
|-----|-------------------|
| **Director/Admin** | Estrategia, captación, negociación, legal, supervisión CRM |
| **Agente comercial** | Visitas, seguimiento leads, captación, cierres |
| **Marketing (opcional)** | Redes sociales, anuncios, contenido blog, SEO |

### Principios operativos:

- Toda comunicación con clientes queda registrada en el CRM
- Cada propiedad tiene un agente responsable asignado
- Los leads se responden en menos de 5 minutos en horario laboral
- Se actualiza el estado de cada expediente semanalmente

## 2. Proceso completo de captación

### 2.1 Lead de captación (propietario que quiere vender/traspasar)

**Canales de entrada:**
- Formulario web "Vende tu propiedad"
- Llamada telefónica entrante
- Referido de otro cliente
- Captación proactiva (puerta fría, redes)

**Proceso:**

| Paso | Acción | Responsable |
|------|--------|-------------|
| 1 | Recibir solicitud y registrar lead en CRM | Admin |
| 2 | Llamar en < 5 min para cualificar | Agente |
| 3 | Concertar visita al local/negocio | Agente |
| 4 | Visita: tomar fotos, medir, evaluar estado | Agente |
| 5 | Presentar propuesta de honorarios y condiciones | Agente |
| 6 | Firmar Dossier de Encargo (sección 3) | Agente + Propietario |
| 7 | Dar de alta la propiedad en CRM y web | Admin |
| 8 | Publicar en redes y anuncios | Marketing |

### 2.2 Captación proactiva (puerta fría)

**Guion telefónico:**

> "Buenos días, soy [nombre] de CENTRAL DE TRASPASOS. Estamos especializados en la venta y traspaso de negocios en [zona]. Llamaba porque tenemos actualmente [X] compradores interesados en [tipo de negocio] en esta zona, y quería saber si estaría abierto a escuchar una propuesta sin compromiso para valorar su local/negocio."

**Kit de captación (llevar a la visita):**
- Dossier de Encargo impreso (2 copias)
- Tarjetas de visita
- Dispositivo para fotos
- Cinta métrica
- Plantilla de toma de datos

## 10. Expediente interno y CRM

### 10.1 Estado del expediente

|--------|-------------|
| **Nuevo** | Lead captado, sin contactar |
| **Contactado** | Llamada realizada, pendiente de visita o respuesta |

| **En negociación** | Oferta recibida o en curso |
| **Reservado** | Arras firmadas, pendiente de formalización |
| **Cerrado** | Operación completada |
| **Cancelado** | No se concreta (archivar con motivo) |

|-------|--------|-------|--------------|--------------|
| | | | | |
| | | | | |

### 10.3 Uso del CRM (data.js / Netlify Blobs)

**En producción (centraldetraspasos.com):**
- Las propiedades se almacenan en Netlify Blobs
- El estado se actualiza desde el panel CRM

**En local:**
- Las propiedades se guardan en localStorage
- Si no hay datos, se cargan las DEFAULT_PROPERTIES (7 propiedades de relleno)

**Reglas:**
- Marca la propiedad como "draft" mientras no esté lista para publicar
- Cuando esté lista -> "published"
- Si se vende/traspasa -> "sold"

## 12. Proceso de cierre y cobro de honorarios

### 12.1 Pasos para el cierre

1. Confirmar que toda la documentación legal está en regla (Checklist Legal)
2. Preparar contrato de arras o contrato de compraventa/traspaso
3. Acordar fecha de firma con todas las partes
4. Confirmar asistencia del notario (si aplica)
5. Preparar Acta de Entrega
6. Cobrar honorarios

### 12.2 Honorarios

| Modalidad | Cuándo se cobra | Cuánto |
|-----------|-----------------|--------|
| Exclusiva | A la firma de arras o escritura | % pactado |
| No exclusiva | A la firma de arras o escritura | % pactado |
| Gestión comprador | A la firma del traspaso | 1 mensualidad (estándar) |

### 12.3 Frase para cobrar (guión)

> "Tal como acordamos en el Dossier de Encargo, en el momento de la firma procederemos al cobro de los honorarios. Puede ser mediante transferencia o recibimos el pago en el momento. ¿Qué opción prefiere?"

### 13.3 Copywriting para posts

**Estructura del texto:**

1. **Gancho** (primera línea que llama la atención)
2. **Datos clave** (precio, ubicación, metros)
3. **Beneficio** (por qué es buena oportunidad)
4. **CTA** (llamada a la acción)

**Ejemplo gancho:**
> "¿Quieres montar tu propio restaurante sin pagar 100.000€?"

**Hashtags obligatorios:**
#Traspaso #Málaga #CostaDelSol #Emprender #CentralDeTraspasos + específicos del tipo de negocio

### 13.4 Anuncios de pago (Facebook Ads)

| Tipo de campaña | Cuándo usarla | Presupuesto mínimo |
|-----------------|---------------|-------------------|
| Tráfico web | Propiedades destacadas | 3-5€/día |
| Leads | Ofertas con precio atractivo | 3-5€/día |
| Mensajes | Negocios de bajo coste (<30k) | 3€/día |
| Interacción | Posts orgánicos con buen rendimiento | 2€/día |

**Checklist para crear anuncio:**
- [ ] Imagen o vídeo de alta calidad
- [ ] Texto con gancho + beneficio + CTA
- [ ] Segmentación: ubicación (ciudad + 15 km) + edad + intereses
- [ ] Enlace correcto a la propiedad
- [ ] Formulario de leads con solo 2-3 campos
